ansonpersona.org is an example primary identity provider for Mozilla Persona. It will let you create (non-functional) email addresses that can be used to log into web sites that use Mozilla Persona.

To get your very own ansonpersona.org address so you can try it out, just sign in.

You can learn more about this feature on Mozilla's Identity blog, review the implementation of ansonpersona.org on github, or get the full details in the documentation.

To sign into an existing ansonpersona.org account, or get a new one, just provide a username and password:
