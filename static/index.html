ansonpersona.org
BrowserID Optimized email addresses

ansonpersona.org is an example "BrowserID primary" - that is, a web site that you can get a (non-functional) email address that is optimized for use with BrowserID.

This is not a real product. It is just an example to show people how they can build first-class BrowserID support into their existing email product.

To get your very own ansonpersona.org address so you can try it out, just sign in.

You can learn more about this feature on our blog, review the implementation of ansonpersona.org on github, or get the full details in the docs.
